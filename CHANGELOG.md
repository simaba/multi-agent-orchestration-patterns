# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] — Foundation release

### Added

- Multi-agent orchestration pattern catalog covering routing, delegation, validation, and failure-handling patterns.
- Reusable orchestration pattern-card template.
- Filled confidence-gated fallback pattern example.
- Pattern guidance for control flow, decision rules, failure modes, logging requirements, and evaluation criteria.
- README guidance clarifying the relationship between this repo, `multi-agent-governance`, `agent-eval`, and `agent-simulator`.

### Notes

This repository focuses on control-flow design. It does not provide the governance operating model, evaluation schema, or executable simulator.

### Next

- Add more filled pattern cards for classifier routing, hierarchical delegation, validator agent, circuit breaker, and human escalation patterns.
- Add a simple mapping from pattern cards to simulator scenarios.
- Add lightweight diagrams for common orchestration flows.
