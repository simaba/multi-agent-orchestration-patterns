# Contributing

Thanks for your interest in improving this repository.

## Scope

This repository documents practical orchestration patterns for multi-agent systems. Helpful contributions improve clarity, reuse, and design quality.

## Good contributions

Examples of helpful contributions include:
- strengthening pattern explanations,
- improving diagrams,
- adding realistic examples,
- clarifying trade-offs between patterns,
- correcting inaccuracies.

## Contribution expectations

Please keep contributions:
- concise,
- implementation-aware,
- free of proprietary or confidential material,
- grounded in practical system design.
